2. Hướng dẫn sử dụng Python cơ bản¶

2.1. Cơ bản về Python¶

Các cấu trúc dữ liệu của Python: - Numbers Integer, long, float, complex - Boolean: T/R - Strings - Lists: Tương ứng với list trong R - Tuples: Vector có giá trị không thể thay đổi được - Dictionary: Cấu trúc dữ liệu hai chiều có key và value

Các thư viện thường dùng trong Python khi phân tích dữ liệu:

  • Numpy: Tính toán đại số
  • Scipy: Tính toán các chỉ số khoa học
  • Matplotlib: Vẽ biểu đồ
  • Pandas: Tương ứng với data.table, dplyr, reshape2
  • Scikit Learn: Machine learning

2.3. Các vấn đề khác¶

2.3.1. Điều kiện¶

  • Sử dụng if, elif hoặc else
[1]:

a = 5
b = 10
if a >= b:
    print("a is larger than b")
elif a == b:
    print("They are equal")
else:
    print("a is smaller than b")

a is smaller than b

2.3.2. Hàm¶

2.3.2.1. Hàm cơ bản¶

  • Sử dụng cấu trúc def - tương tự với R
[2]:

def from_city(city = "Hanoi"):
    print("I am from " + city)
from_city()

I am from Hanoi
  • Hàm trả ra giá trị: sử dụng return
[3]:

def square(x):
    return(x**2)

[4]:

square(4)

[4]:

16

[5]:

y = square(9)
print(y)

81
