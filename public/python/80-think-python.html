12. Think Python¶

12.1. Giới thiệu¶

  • Ngôn ngữ tự nhiên (natural language) là ngôn ngữ do con người sáng tạo qua nhiều năm để nghe, nói, đọc, viết.
  • Formal language là ngôn ngữ do con người tạo ra để phục vụ các ứng dụng nhất định - Token là đơn vị của ngôn ngữ. Ví dụ: 3*8=1&8, & không phải là token được sử dụng trong phép toán - Structure là cấu trúc hay ngữ pháp của ngôn ngữ (syntax)
  • Khi đọc 1 statement, ta cần phải tìm hiểu cấu trúc của statement đó, quá trình này gọi là parsing

Trong lập trình, có 3 nhóm hàm lớn là reduce, map và filter

  • Quá trình tích hợp chuỗi các phần tử thành 1 giá trị được gọi là reduce.
  • Quá trình áp dụng một hàm vào từng phần tử của list được gọi là map
  • Quá trình lựa chọn các phần tử theo điều kiện được gọi là filter

def my_func(x):

12.7. Class vs. Methods¶

Methods là hàm được sử dụng với một số class nhất định. Bản chất của methods là hàm nhưng có các điểm khác biệt sau

  • Methods được khai báo ngay trong class
  • Ngữ pháp để gọi methods sẽ khác so với với gọi function
[147]:

# Cách viết thường
class Time:
    """Trả ra kết qua time"""

def print_time(time):
    print('%.2d:%.2d:%.2d' % (time.hour, time.minute, time.second))

[149]:

start = Time()

[151]:

# Assign attribute
start.hour = 9
start.minute = 45
start.second = 20
print_time(start)

09:45:20

Với cách viết trên, ta phải để time dưới dạng argument. Tuy nhiên, ta có thể thay đổi cách viết bằng cách tạo methods trong class như sau.

[166]:

12.8. Init method¶

  • __init__ được sử dụng khi tạo ra object đầu tiên trong class
  • selft là method được sử dụng khi tạo ra đối tượng mới (instance)
[174]:

class Time:
    def __init__(self, hour = 0, minute = 0, second = 0):
        self.hour = hour
        self.minute = minute
        self.second = second
    def print_time(time):
        print('%.2d:%.2d:%.2d' % (time.hour, time.minute, time.second))

[175]:

start = Time()

[178]:

start.print_time()

Khi sử dụng class với init, ta có thể xây dựng tham số

12.9. __str__ method¶

  • __str__ là method cho phép hiển thị giá trị của instance
[186]:

class Time:
    def __init__(self, hour = 0, minute = 0, second = 0):
        self.hour = hour
        self.minute = minute
        self.second = second
    def print_time(time):
        print('%.2d:%.2d:%.2d' % (time.hour, time.minute, time.second))
    def __str__(time):
        return '%.2d:%.2d:%.2d' % (time.hour, time.minute, time.second)

[188]:

print(Time(4,5,8))

04:05:08

[189]:

Time(4,5,6).print_time()

04:05:06
